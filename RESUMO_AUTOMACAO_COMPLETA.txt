📞 SE DER ERRO
════════════════════════════════════════════════════════════════════════════════

❌ "npm não encontrado"
   → Instale Node.js: https://nodejs.org/
   → Reinicie computador

❌ "Permission denied" (PowerShell)
   → Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   → Depois execute novamente

❌ "Build falhou"
   → Verifique erros no console
   → Deleta node_modules + package-lock.json
   → Execute: npm install

❌ "Chave reCAPTCHA não funciona"
   → Verificar que é Site Key (não Secret)
   → Está em .env.local?
   → Está no Lovable Settings?
   → Limpar cache: Ctrl+Shift+Delete
